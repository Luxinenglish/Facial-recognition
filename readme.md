### Comment utiliser la reconnaissance facial ?
----
Prérequis : Python 3 

Cette commande va installer toutes les dépendances necaissaire :
```sh
pip install -r ./requirements.txt
```

Puis executer l'app :
```sh
python3 main.py
```

Puis Rendez-vous à l'adresse : http://127.0.0.1:80
----
